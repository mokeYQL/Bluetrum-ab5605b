# ESP32S3 ↔ AB5605B 串口通信协议

## 协议约定

| 参数 | 值 |
|------|------|
| 接口 | UART, TTL 3.3V, 全双工 |
| 波特率 | 115200 bps, 8N1 |
| 字节序 | 大端 (Big-Endian) |

## 数据格式

`[0xA5][命令][LEN_H][LEN_L][数据段...][FCS]`

- **FCS** = CMD ^ LEN_H ^ LEN_L ^ DATA[0] ^ ... ^ DATA[N-1] (异或校验)
- 主控(ESP32S3)发送 / 从控(AB5605B)应答，对应命令码相同

## 命令定义

| 命令 | 功能 | 主控发送 | 从控应答 |
|:---|:---|:---|---:|
| `0x01` | 查询设备信息 | `A5 01 00 00 [FCS]` | `A5 01 00 N [JSON] [FCS]` |
| `0x02` | 系统复位 | `A5 02 00 00 [FCS]` | 无（立即复位） |
| `0x03` | 静音控制 | `A5 03 00 01 [00=取消/01=静音] [FCS]` | `A5 03 00 01 00 [FCS]` |
| `0x04` | 音量 0~16 | `A5 04 00 01 [0x00~0x10] [FCS]` | `A5 04 00 01 00 [FCS]` |
| `0x05` | 音频切换 | `A5 05 00 01 [01=BT/02=AUX] [FCS]` | `A5 05 00 01 00 [FCS]` |
| `0x06` | 蓝牙状态查询 | `A5 06 00 00 [FCS]` | `A5 06 00 01 [00断开~03播放中] [FCS]` |
| `0x07` | LED灯效 | `A5 07 00 01 [00关/01亮/02呼吸/03律动] [FCS]` | `A5 07 00 01 00 [FCS]` |
| `0x08` | 广播名称 | `A5 08 00 N [UTF-8名称] [FCS]` | `A5 08 00 01 00 [FCS]` |

> 查询设备信息应答 JSON：`{"pn":"产品名","fn":"厂家","hv":"x.x.x","sv":"x.x.x"}`

## OTA语音升级协议

批量更新 AB5605B 语音提示音资源（MP3/TOC），写入 NOR Flash `0x60000`，分区大小 `0x1B000`（108KB），数据块 256 字节。

### 命令

| 命令 | 功能 | 主控发送 | 从控应答 |
|:---|:---|:---|---:|
| `0x09` | 擦除 | `A5 09 00 02 [crc16_lo][crc16_hi] [FCS]` | `A5 09 00 01 00 [FCS]` |
| `0x0A` | 写数据 | `A5 0A 01 03 [seq 2B LE][xor][256B data] [FCS]` | `A5 0A 00 02 [seq 2B LE] [FCS]` |
| `0x0B` | 完成校验 | `A5 0B 00 00 [FCS]` | `A5 0B 00 01 [00通过/01失败] [FCS]` |

- **擦除**：擦除全部分区，保存 CRC16-CCITT 值供完成校验使用
- **写数据**：seq 从 0 递增（小端），xor = 256 字节数据的异或和，不匹配时不回 ACK
- **完成校验**：读回 Flash 计算 CRC16，通过则自动复位

### 完整流程

```
ESP32S3                                    AB5605B
  │ A5 09 00 02 [crc_lo][crc_hi] [FCS]       │
  │─────────────────────────────────────────→│  → 擦除完成
  │←─ A5 09 00 01 00 [FCS] ──────────────────│
  │                                            │
  │ A5 0A 01 03 [seq][xor][256B] [FCS] (循环) │
  │─────────────────────────────────────────→│  → 写入 Flash
  │←─ A5 0A 00 02 [seq] [FCS] ───────────────│
  │                                            │
  │ A5 0B 00 00 [FCS]                         │
  │─────────────────────────────────────────→│  → CRC校验 → 复位
  │←─ A5 0B 00 01 00 [FCS] ──────────────────│
```

### 算法参考

**XOR**：`xor = 0; for each byte: xor ^= byte`

**CRC16-CCITT**：多项式 `0x1021`，初始 `0x0000`

## FCS 计算

```c
u8 calc_fcs(u8 cmd, u8 *data, u16 len) {
    u8 fcs = cmd ^ (len >> 8) ^ (len & 0xFF);
    for (u16 i = 0; i < len; i++) fcs ^= data[i];
    return fcs;
}
```
