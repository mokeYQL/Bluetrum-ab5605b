# ESP32S3 ↔ AB5605B 串口通信协议

## 协议约定

| 参数 | 值 |
|------|------|
| 通讯方式 | 串行异步通讯 (UART) |
| 接口电平 | TTL / LVTTL (3.3V) |
| 传送方式 | 全双工 |
| 波特率 | 115200 bps |
| 数据位 | 8 bit |
| 停止位 | 1 bit |
| 校验位 | 无 (帧级校验和保证完整性) |
| 字节序 | 大端模式 (Big-Endian) |

## 数据格式

| 类型 | 起始码（uint8） | 命令（uint8） | 长度（uint16） | 数据段                     | FCS（uint8） |
| :--- | :-------------- | :------------ | :------------- | :------------------------- | :----------- |
| 描述 | 帧头0xA5        |               | 数据段的长度   | 不同命令的数据段长度会不同 | 异或校验     |

## 命令说明

| 命令   | 含义       | 方向             |
| :----- | :--------- | :--------------- |
| `0x21` | 主机->从机 | AB5605 → ESP32S3 |
| `0x20` | 从机->主机 | ESP32S3→AB5605   |

## 功能点

| 名称 | 功能 |
|------|------|
| 查询设备信息 | 获取硬件版本、软件版本、产品名 |
| 系统控制 | 复位 |
| 静音控制 | 静音/取消静音 |
| 音量级别 | 0~16 |
| 音频切换 | `0x01`:BT-A2DP, `0x02`:AUX |
| 蓝牙连接状态 | `0x00`:断开, `0x01`:连接中, `0x02`:已连接, `0x03`:播放中 |
| 语音OTA状态 | `0x00`:成功, `0x01`:失败                                 |
| LED灯效 | `0x00`:关闭, `0x01`:常亮, `0x02`:呼吸, `0x03`:音乐律动 |
